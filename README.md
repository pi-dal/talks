## 🛠️ Development

### Setup
```bash
# Install dependencies
pnpm install

# Organize talks by metadata
pnpm run organize

# Start development server
pnpm run dev
```

### Build & Deploy
```bash
# Build all talks
pnpm run build:all

# Deploy to Cloudflare Pages
# (Automatically triggered on git push)
```
